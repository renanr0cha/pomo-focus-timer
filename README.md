
# Pomo Focus Timer

A timer to use when doing pomodoro sprints, while listening to calming background music.

- Light/dark mode toggle
- DOM Manipulation
- ES6 Modules
- Declarative programming
- Clean Code
- Events
- Callbacks
- Factories
- Dependency Injection
